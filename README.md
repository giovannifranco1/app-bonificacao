# Bonus

### Tecnologias

- Backend: Laravel, Php 8
- Frontend: Laravel - Blade 

### Como executar

- Requisitos:
  - Php8
  - Mysql

Clone o projeto em seu dispositivo, Abra o terminal,  execute `php artisan serve`,
a aplicação irá rodar no endereço: `http://localhost:8000`
